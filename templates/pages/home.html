よこそう

動詞活用 (Verb Conjugation)

Japanese Aspect: Is a state that exists as a result of entering that state. Meaning, if something is still on going you are in the result. It actively resulting in the state.

  • 走っている = "is in the state of having started running" (we translate as "is running")
  • 死んでいる = "is in the state of having died" (we translate as "is dead")
  • 知っている = "is in the state of having come to know" (we translate as "knows")

Therefore the continuous and the resultant both typically present using ている. Because of this selecting resultant will attempt to use てある which could be considered the perfect or resultative perfect aspect.

True Japanese auxiliary verbs 助動詞 (Jodōshi): Japanese makes the distinction with these suffixes as being the auxiliary verbs, however it can be beneficial to treat them as the base form of a conjugated verb. For example: れる/られる is an auxiliary verb to make a base verb in the passive or potential form. So 食べる can go from "eats" to 食べられる "can eat"
